Título do Projeto:
    
    Adivinhe o Número Secreto

Nível:
    
    Iniciante

Objetivo:
    
    Criar um jogo simples onde o computador "pensa" em um número inteiro entre 1 e 100, e o jogador tem que adivinhar qual é esse número.
    O programa dará dicas se o palpite do jogador foi muito alto ou muito baixo.

Conceitos Chave de Python a serem aplicados:

    Uso da biblioteca random para gerar números aleatórios.
    Entrada de dados do usuário com a função input().
    Laços de repetição (while).
    Estruturas condicionais (if, elif, else).
    Conversão de tipos de dados (int()).

Requisitos/Funcionalidades:

    O programa deve gerar um número secreto aleatório no início do jogo.
    O programa deve pedir ao jogador para digitar um palpite.
    Após cada palpite, o programa deve informar ao jogador se o número secreto é maior ou menor que o palpite dado.
    O jogo deve continuar pedindo palpites até que o jogador acerte o número.
    Quando o jogador acertar, o programa deve exibir uma mensagem de parabéns e encerrar.

Desafio Extra (Opcional):

    Conte e exiba o número de tentativas que o jogador levou para acertar.
    Permita que o jogador escolha o intervalo de números no início do jogo (por exemplo, de 1 a 50 em vez de fixo em 1 a 100).
